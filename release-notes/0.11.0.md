# Pixel Splash Studio 0.11.0

## Highlights
- New top bar + reorganized tool UI (tool groups, context menus, bottom dock controls).
- New **Magic Wand** selection tool.
- New **Text** tool with improved clipboard handling.

## New
- Magic Wand tool for selecting areas by color/threshold.
- Text tool and `TextToolModal` UI for editing/applying text.
- Centralized hotkey handling (`src/renderer/services/hotkeys.ts`) with better test coverage.
- Additional UI components: `Topbar`, `ToolGroups`, `ToolContextMenu`, `BottomDockControls`, and shared tool icons.

## Improvements
- Lasso selection tool behavior and related UI/clipboard flows.
- Clipboard workflows for selections and text (new selection/text clipboard services).
- Palette UI and ramp/consolidation flows.
- Viewport canvas/input handling and general renderer polish.

## Fixes
- Pressing **Delete/Backspace** now clears selected pixels.
- Fixed a production build issue caused by a duplicate hotkey `action` declaration.

## Dev / QA
- Added and expanded unit tests for hotkeys, magic wand, palette ramp, lasso selection, and clipboard behavior.
