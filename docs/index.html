Pixel art with speed, snap, and soul.

Pixel Splash Studio is a GtkSharp-powered editor built for crisp drawing tools, flexible selection workflows, and reference layers you can bake into pixels when it is time to commit.

Get the latest build Build from source
Fast tools + tight grid control
Built for pixel-first workflows

Precise tools

Pen, line, rectangles, ovals, erase, flood fill, and a stamp tool with rotate, flip, scale, snap, and overwrite modes.

Selection power

Rectangle, oval, magic wand, plus add and subtract modes so you can stay in control of every pixel.

Reference layers

Paste or load references, adjust opacity, and bake them into your pixel layer when you are ready.

Configurable shortcuts

Shortcut bindings live in `config.ini`, making it easy to tailor the workflow to your muscle memory.

How the layers flow

Pixel layer

Core drawing surface for every tool, with grid-aware snapping and fast selection editing.

Selection layer

Add, subtract, and refine before committing. Export selections directly to PNG when needed.

Reference layer

Drag in images or text, adjust opacity, then bake into pixels to keep the source nearby.

Custom file format

Save `.pss` projects to keep every layer and tool setting intact between sessions.

Downloads and builds

Linux AppImage

Run `scripts/build_linux_appimage.sh` to produce `dist/pixel-splash-studio.AppImage`. Requires GTK 3 runtime and `appimagetool`.

Windows packages

Run `scripts/build_windows_packages.ps1` for zip packages (and optional self-extracting executables with 7z installed).

From source

`dotnet build` for a quick local build, then `dotnet run` to jump straight into the editor.
